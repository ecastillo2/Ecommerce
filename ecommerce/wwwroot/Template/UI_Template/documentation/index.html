About the theme

Orise Store is a multi-purpose ecommerce template for every type of website, including 20+ different page layouts to build your own website.

Orise Store HTML5 Template can be used for multi-purpose websites. Wether you want to create a ecommerce shop, a business website, a blog or even a personal portfolio, Orise Store is always the right solution. Orise Store is a template for fashion shops and ecommerce websites. It is a highly suitable template for all types of ecommerce. It has flat and purpose oriented design, responsive layout and special features like appointment forms, checkout pages, shopping carts, login forms, popup windows and other shop related elements, as well as blog layouts and landing pages. Use easy theme options and shortcodes to customize your website. Orise Store theme is powered by Twitter Bootstrap v3 (the most popular front-end framework). It means that it will be easier for you to do the changes and to increase its possibilities, thanks to many plugins it possesses.

Installation

Orise Store - eCommerce HTML5 Template does not need a real installation. Just upload all files included in folder ./Orise Store to your server.

Upload Template files

You have two ways to upload Orise Store Template files:

  1. Upload Orise Store files in folder ./Orise Store to your server via FTP
    • Unpack the Template .zip file (OriseStore_ThemeForest*.zip) you have downloaded.
    • Upload the /OriseStore folder on your server.

Files

Orise Store Template includes many files, within which you'll find the following directories and files, logically grouping common resources and providing both finding and changes.

Contact Form

You need to modify email address in Contact.php on line 51 to your email address. So if the user send a message using contact form, it will go straight to your email.

$to = 'rafiquejadoon2@gmail.com';

Note that Your server must support php send mail function to make contact form work.

Contact map

In order to show your location on Google map, You need to modify your location in googlemap-init.js. Change the following values:

"lat": '40.730610',
"lng": '-73.935242',

Note that Your server must support php send mail function to make contact form work.
